# Offline Launcher

Офлайн-лаунчер для Minecraft с поддержкой загрузчиков (Forge, Fabric, Quilt), скинов и модов.

## Требования

- **Java 17+** ([скачать](https://adoptium.net/))
- **Операционная система**: Windows 10+, Linux, macOS
- **Интернет**: требуется только для загрузки версий и обновлений

## Установка

1. [Скачайте](https://github.com/user/offline-launcher/releases) последний релиз для вашей ОС
2. Распакуйте архив в любую папку
3. Убедитесь, что Java 17+ установлена:
   ```bash
   java -version
   ```
4. Запустите лаунчер:
   - **Windows**: `launcher.exe`
   - **Linux/macOS**: `./launcher`

Лаунчер откроется в браузере по адресу `http://localhost:8080`

## Быстрый старт

1. Введите никнейм и нажмите **"Войти офлайн"**
2. Перейдите на вкладку **"Версии"**, нажмите **"Установить"**
3. Выберите версию Minecraft из списка
4. (Опционально) Установите загрузчик на вкладке **"Моды"**
5. Вернитесь на **"Главную"**, выберите версию и нажмите **Play**

## Настройка

### Память
На вкладке **"Настройки"** можно изменить:
- **Xms** (минимальная ОЗУ): 512–8192 MB
- **Xmx** (максимальная ОЗУ): 1024–16384 MB

### Путь к Java
Если Java не найдена автоматически, укажите полный путь:
- **Windows**: `C:\Program Files\Java\jdk-17\bin\java.exe`
- **Linux/macOS**: `/usr/lib/jvm/java-17-openjdk/bin/java`

## Скины и плащи

1. Войдите в аккаунт на главной
2. Нажмите **"Загрузить скин"** / **"Загрузить плащ"**
3. Выберите PNG-файл
   - Скин: 64×32 или 64×64 пикселя
   - Плащ: любые PNG
4. Скин автоматически подставится при запуске через authlib-injector

## Установка модов

### Modrinth
1. На вкладке **"Моды"** нажмите **"Найти"**
2. Введите название, выберите загрузчик
3. Нажмите **"+"** рядом с модом

### CurseForge
1. На вкладке **"Настройки"** введите CurseForge API Key
2. Используйте поиск на вкладке **"Моды"**

## Установка загрузчиков

1. На вкладке **"Моды"** нажмите **"Установить"** под загрузчиками
2. Введите версию Minecraft (например, `1.20.4`)
3. Выберите загрузчик: forge / fabric / quilt

Лаунчер автоматически скачает инсталлятор и создаст профиль.

## Обновление

Лаунчер проверяет обновления при запуске. При наличии новой версии:
1. Нажмите **"Обновить"** в карточке обновлений (вкладка "Моды")
2. Дождитесь загрузки и перезапуска
3. В случае ошибки старый файл сохраняется как `launcher.exe.old`

## Сборка из исходников

```bash
# Клонирование
git clone https://github.com/user/offline-launcher.git
cd offline-launcher

# Сборка
make build

# Запуск
make run

# Тесты
make test

# Сборка под все платформы
make pack-all
```

## Структура проекта

```
cmd/launcher/       # Точка входа
internal/
├── auth/           # Управление аккаунтами
├── authserver/     # Fake Mojang Yggdrasil сервер
├── download/       # Загрузка версий Minecraft
├── installid/      # ID инсталляции
├── launch/         # Запуск игры (аргументы, процессы)
├── log/            # Логирование с ротацией
├── mods/           # Моды и загрузчики
├── profiles/       # launcher_profiles.json
├── server/         # REST API (gin)
├── settings/       # Настройки лаунчера
├── skin/           # Скины и плащи
└── update/         # Самообновление
web/                # Фронтенд (SPA)
```

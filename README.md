# Go Launcher

Легкий, быстрый и красивый офлайн-лаунчер для Minecraft. Один .exe — всё встроено.

**Go не нужен!** Лаунчер — это готовый .exe. Он работает на чистой Windows, Linux и macOS без установки Go или любого другого языка программирования. Java нужна только для запуска Minecraft.

## Возможности

- **Один файл** — веб-интерфейс вшит в бинарник через //go:embed. Никаких установщиков.
- **Автооткрытие** — запускает браузер сам.
- **Управление версиями** — установка и запуск любых версий Minecraft.
- **Загрузчики** — Forge, Fabric, Quilt одним кликом.
- **Моды** — поиск и установка с Modrinth и CurseForge. Профили модов без дублирования файлов.
- **Офлайн-режим** — вход по нику, встроенный authlib для скинов.
- **Скины и плащи** — загрузка PNG, автоматическая установка на сервере.
- **Автообновления** — проверка новых версий на GitHub.
- **Тёмная тема** — современный интерфейс с Bootstrap Icons.

## Требования

- **Java 17+** ([скачать](https://adoptium.net/))
- **ОС**: Windows 10+, Linux, macOS
- **Интернет**: только для загрузки версий Minecraft и обновлений

## Быстрый старт

1. Скачай и запусти go-launcher.exe — откроется браузер.
2. Введи никнейм и нажми **«Войти офлайн»**.
3. Перейди на вкладку **«Версии»** → **«Установить»** и выбери версию.
4. (Опционально) Установи Forge/Fabric/Quilt на вкладке **«Моды»**.
5. На **«Главной»** выбери версию и нажми **Play**.

> Чтобы **полностью закрыть** лаунчер, нажми кнопку ✕ в левом нижнем углу.
> Чтобы открыть снова — просто запусти go-launcher.exe.

## Настройки

### Память

На вкладке **«Настройки»**:
- **Xms** (мин. ОЗУ): 512–8192 MB
- **Xmx** (макс. ОЗУ): 1024–32768 MB

### Путь к Java

Если Java не найдена автоматически, укажи вручную:
- **Windows**: C:\Program Files\Java\jdk-17\bin\java.exe
- **Linux**: /usr/lib/jvm/java-17-openjdk/bin/java
- **macOS**: /Library/Java/JavaVirtualMachines/jdk-17/Contents/Home/bin/java

## Структура

`
cmd/launcher/           # Точка входа, embed веб-файлов
├── main.go
├── web/                # Фронтенд (встраивается в бинарник)
│   ├── index.html      # SPA — HTML, CSS, JS в одном файле
│   └── icon.svg        # Иконка / фавикон

internal/
├── auth/               # Управление офлайн-аккаунтами
├── authserver/         # Встроенный Yggdrasil-сервер
├── download/           # Загрузка версий Minecraft
├── installid/          # ID установщика
├── launch/             # Запуск игры (аргументы, процессы)
├── log/                # Логирование с ротацией
├── mods/               # Моды (Modrinth / CurseForge)
├── open/               # Открытие браузера (кросс-платформа)
├── profiles/           # Профили запуска
├── server/             # REST API (gin)
├── settings/           # Настройки
├── skin/               # Скины и плащи
└── update/             # Проверка и установка обновлений
`

## Обновления через GitHub

Лаунчер автоматически проверяет новые релизы на GitHub.
При обнаружении новой версии вверху страницы появляется баннер:

- **«Скачать»** — переход на страницу релиза
- **«Позже»** — скрыть уведомление

Версия задаётся в internal/update/update.go (константа CurrentVersion).
Тег релиза на GitHub сравнивается с ней (префикс  игнорируется).

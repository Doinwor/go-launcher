Go Launcher

Minecraft Offline Launcher

Лёгкий, быстрый и красивый лаунчер для Minecraft. Управляй версиями, загрузчиками и модами в одном окне.

Скачать v1.0.8 GitHub

Возможности

Всё необходимое для игры

Go Launcher объединяет управление версиями, загрузчиками и модами в едином интерфейсе.

⚙
Управление версиями
Устанавливай любые версии Minecraft из официального манифеста. Быстрая установка и переключение.
⚙
Загрузчики
Forge, Fabric, Quilt — устанавливай и переключайся между ними. Каждая версия со своим загрузчиком.
⚙
Профили модов
Создавай профили модов для каждой версии. Включай и отключай моды без удаления.
